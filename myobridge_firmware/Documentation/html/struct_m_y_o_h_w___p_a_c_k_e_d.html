Detailed Description

Header that every command begins with.

Classifier event data received in a myohw_att_handle_classifier_event attribute.

Motion event data received in a myohw_att_handle_motion_event attribute.

Integrated motion data.

User action command.

Set sleep mode command.

Deep sleep command.

Vibration command.

Command to set EMG and IMU modes.

Various parameters that may affect the behaviour of this Myo armband. The Myo library reads this attribute when a connection is established. Value layout for the myohw_att_handle_fw_info attribute.

Version information for the Myo firmware. Value layout for the myohw_att_handle_fw_version attribute. Minor version is incremented for changes in this interface. Patch version is incremented for firmware changes that do not introduce changes in this interface.

Unlock Myo command. Can also be used to force Myo to re-lock.

Raw EMG data received in a myohw_att_handle_emg_data_# attribute. Value layout for myohw_att_handle_emg_data_#.
